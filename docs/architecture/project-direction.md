# Project Direction

Status: framework baseline

## Purpose

V4 is a civil estimating platform built around drawing intake, quantity takeoff, optional costing, and strict project-instance isolation.

## Modes

1. Quantity takeoff only.
2. Quantity takeoff plus optional cost buildout.

Quantity takeoff is a valid standalone deliverable. Costing is an optional escalation from reviewed takeoff data.

## Phase 1

- First work type: PVC gravity sewer.
- Slice 1A: drawing to reviewed quantity takeoff.
- Slice 1B: reviewed takeoff to optional cost buildout.
- Pipe depth allocation basis: ground or profile to pipe invert.

## Framework requirement

The platform must remain a reusable generic framework. Project data, sandbox data, sample jobs, overrides, and snapshots must not become framework baseline content by default.
